数组的总结: 数组:存储一组有顺序的数据; 数组的作用:一次性存储多个数据; 数组的定义方式: 1:构造函数定义数组 var 数组名 = new Array(数据); var 数组名 = new Array(); 定义了一个空数组,长度为0 var 数组名 = new Array(一个数字); 定义了一个数组.数组长度就是这个数字 var 数组名 = new Array(值1,值2,值3...); 定义了一个数组并且有多个数据 2:字面量定义数组 var 数组名 = [数据]; var 数组名 = []; 定义了一个空数组 var 数组名 = [值1]; 定义了一个数组,数组长度为1,数据为值1 数组元素:数组中存储的数据; 数组长度:数组中存储数据的个数; 数组索引(下标):从0开始,直到数组的个数-1 通过下标访问数组的元素值: 数组名 [下标]; 通过下标设置数组的元素值: 数组名 [下标] = 值;
