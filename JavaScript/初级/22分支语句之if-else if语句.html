if语句: if语句只有一个分支 if-else语句: if语句只有两个分支,最终执行一个分支 if-else if-else if-else if....else 多个分支,但最终也是只执行一个分支 语法: if(表达式){ 代码块1 }else if(表达式2){ 代码块2 }else if(表达式3){ 代码块3 }else(表达式4){ 代码块4.... }.... 执行顺序: 先执行表达式1,如果结果为true,执行代码块1,结果为false,则执行表达式2 如果表达式2结果为true,执行代码块2,结果为false,则执行表达式3 如果表达式3结果为true,执行代码块2,结果为false,则执行表达式4 如果表达式4结果为true,执行代码块3,结果为false,则执行表达式5 ....以此类推 else if 这种结构可以写多个,看需求 else 结构是可以不用写的,看需求
