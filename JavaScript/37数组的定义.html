数组的定义:一组有序的数据 作用:可以储存多个变量 1.通过构造函数创建数组 语法: var name = new Array(); var 数组名 = new Array(长度) var arr = new Array(); 打印出这个数组,如果数组中没有数据,但是有长度---,数组中每个的值就是undefined 2.通过字面量创建数组 var arr = []; 通过字面量的方式创建一个空数组 重点:无论是构造函数还是字面量创造的数组,如果没有长度,那么默认是undefined 数组:一组有顺序的数据 数组元素:数组中储存的每个数据,都可以叫数组元素,比如:储存了3个数据,就是数组中的3个元素 数组长度:数组中元素的个数,比如有3个元素就是这个数组的长度为3 数组索引(下标):用来存储或者访问数组中的数据的,下标从0开始计算,到长度减1 1.通过构造函数的方式创建一个数组 var arr = new Array(一个数字);----这代表这个数组的长度 如果在Array(写下多个数字) 那个这个数组元素的个数就是数组的长度 数组的长度和数组的下标的关系:下标从0开始计算,到数组的长度减1停止 如何获取数组中某个位置的值 console.log(数组名[下标]); 或者 var result=数组名[下标]; //创建一个变量来接收这个数据再打印 var result=arr[2]; console.log(result); 如何设置一个数组中的某个位置的值 数组名[下标]=值; arr[2]=100;
